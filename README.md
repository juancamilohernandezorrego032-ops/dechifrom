# 🐜 De-Chill — Gestión Inteligente de Gastos Hormiga hol


## 📝 Descripción del Proyecto
**De-Chill** es una plataforma web diseñada para ayudar a los usuarios a identificar, registrar y analizar esos "gastos hormiga" que afectan la salud financiera a largo plazo. Actualizado para cumplir con los estándares de flujo de Git y requisitos visuales del docente.

* **HU-03 — Registro en la app:** Formulario detallado con validación de campos obligatorios para crear cuenta.
